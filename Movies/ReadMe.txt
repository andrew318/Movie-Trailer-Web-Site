Steps to run this Movie Web Site Application

1. Unzip folder.
2. Either right-click fresh_tomatoes.html and select Open with, and then select your browser of choice, or:
open MovieWebSite.py in a code editor with a Python interpreter included and build the application.
To build the application using Sublime Text, with the MovieWebSite.py file open in the editor, press Ctrl + B or go to Tools > Build in the top menu bar.
